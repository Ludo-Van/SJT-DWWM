10. Le CSS

le Cascading Styles Sheets est un langage de style qui permet de mettre en forme les pages HTML, il permet de definir la couleur, la taille, la police de caractere, la mise en page, les animations, les transitions... Les standars definicents le CSS sont publier par le W3C(world whild web consortium)

10.2 les objectifs

  • separer le contenu de la presentation, c'est a dire séparer la structure html des styles CSS
  • controler l'apparence et le rendu des pages html>
  • controler exclusivement la forme et non le fond
  • interpretation et compatibilite avec tous les navigateurs

10.9. le media queries

les medias queries font parties des nouveautées de CSS3. il ne sajit pas de nouvelle proprietées mais de regles que l'ont peut appliquer dans certaines conditions. concrètement nous allons pouvoir dire si la resolution de l'ecran de l'utilisateur est inférieure à tant, applique ces propriétés css.cela nous permet de changer l'apparence du site selon la fonction des tailles de l'ecran. c'est ce qu'on appel le responsive design.

la syntaxte

une requette media se compose d'un type de media optionnel et d'une ou plusieurs expressions de caracteristique de media. Plusieurs requettes peuvent etre combinées entre elles, grace a des opérateurs logique. les requettes médias ne sont pas sensible à la casse.
Une requette media vaut true si le type de media correspond à l'appareil utilisé sinon elle vaut false.

Les types de médias

un type de médias definit une catégorie générale d'appareils, le type de média est optionnel dans une requette media, sauf si celle-ci utilise les opérateurs logique not ou only. Par defaut, le type de média utilisé est ALL.

  • all : applique les styles css à tous les types de médias
  • print : applique les styles css aux impression et aux documents en mode aperçu avant impression(paginer)
  • screen : applique les styles css aux écrans d'ordinateurs, tablettes, smartphones...
  • speech : applique les styles css aux synthèses vocales

Les caractéristiques media

Elles décrivent certaines spécificités de l'agent utilisateur de l'appareil d'affichage ou de l'environement. Les expressions des carractéristique médias testent leur presence ou leur valeur.
Exemple:

  • color
    • gestion de la couleur
  • width
    • gestion de la largeur des ecrans
  • eignt
    • gestion de la hauteur des écrans
  • ...

ont peut rajouter les prefix minet maxdevant la plus part de ces caractéristiques, ainsi min-width voudra dir la largeur minimale.

Opérateur logique

les operateurs logiques not , only et and peuvent etre utiliser afin de construire une requette media complexe. Il est aussi possible de combiner plusieurs requettes media en les separant par des virgules.

  • not
    • Il est utiliser pour obtenir le resultat inverse de la requette média
  • only
    • Il est utiliser afin d'appliquer un style unique à une requette
  • and
    • Il permet de combiner plusieurs requettes médias
  • ,
    • Il permet de combiner plusieurs requettes en une seule

Application

Les médias queries sont des règles qui indiques quand on doit appliquer des propriétés css. Il existe deux manières de les utiliser.

  • En chargeant une feuille de style CSS different en fonction de chaque support.
  • En utilisant les requettes dans le fichier CSS habituel style CSS.
